FIREBASE
SETUP
Before launching, connect your Firebase project. This only takes 5 minutes and is completely free.
1 Create Firebase Project

Go to console.firebase.google.com, click "Add project", name it quilla-hub, disable Google Analytics (optional), click Create.

2 Enable Authentication

In your project → Build → Authentication → Get Started → Sign-in method tab → Enable Email/Password.

3 Create Firestore Database

Build → Firestore Database → Create database → Start in test mode → choose a region → Done.

⚠ Switch to production rules before going live (instructions in the deployment guide).

4 Register a Web App

Project Overview → click the </> icon → nickname: quilla-hub → Register app → copy the firebaseConfig object.

5 Paste your config below

Config is saved in your browser's localStorage — it stays here even after refreshing.
